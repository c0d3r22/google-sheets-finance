# Original Code - Legacy Version

This folder contains the original, unorganized version of the Google Sheets Finance Tracker codebase.

## Contents

- `Code.gs` - Main application logic (sheet creation, expense management, UI)
- `plaidAPI.gs` - Plaid banking API integration
- `accountLinkingPlaid.gs` - Plaid account linking functionality
- `Teller.gs` - Teller.io banking API integration
- `Test.gs` - Test and utility functions
- `htmlDropdownFunctions.gs` - HTML dropdown helper functions
- `modal.html` - Sheet creation modal
- `editExpenses.html` - Expense editing interface
- `editExpensesDropdown.html` - Alternative expense editing interface
- `PlaidLink.html` - Plaid account linking UI
- `README.md` - Original project documentation

## Status

This is the working, original codebase that has been reorganized into the `src/` folder for better maintainability.

## Migration

The code in `src/` has been reorganized into logical modules:
- `src/core/` - Core application logic
- `src/integrations/` - Third-party API integrations
- `src/ui/` - User interface components
- `src/utils/` - Utility functions

The reorganized version maintains the same functionality but with improved structure and maintainability.
